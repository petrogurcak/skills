# Brand Naming Skill — Design Spec

**Date:** 2026-03-21
**Plugin:** branding
**Skill:** brand-naming
**Status:** Approved

## Overview

Complete brand naming workflow: from brief through linguistic analysis to domain-checked finalists. 7-step process combining Watkins SMILE/SCRATCH, Meyerson naming methodology, and advanced linguistics.

## 7-Step Process

1. **Brand Brief** — load brand voice doc or run mini-discovery (audience, personality, tone, competition, evocations)
2. **Name Taxonomy Selection** — 5 types (Descriptive/Suggestive/Evocative/Invented/Abstract), user picks 1-3
3. **Name Generation** — 30-50 names using linguistic techniques (portmanteau, blending, affixation, metaphor, mythology, sound symbolism, clipping)
4. **SMILE/SCRATCH Evaluation** — score + phonetic scoring + filter to top 5-8
5. **Linguistic Deep Analysis** — sound symbolism, morphological decomposition, cross-language screening, pronounceability
6. **Domain Check** — instantdomainsearch MCP (.com, .io, .co, country TLDs)
7. **Final Presentation** — top 3-5 with full profile, domains, recommendation #1

## Integration

- Input: brand-voice document (optional) or mini-discovery
- Output: name + domains + evaluation
- Follows: logo-design skill for visual identity
- Uses: instantdomainsearch MCP
