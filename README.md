**Total:** 37 skills

## Installation

```bash
# Register marketplace
claude plugins marketplace add petrogurcak/skills

# Install plugins
claude plugins install seo
claude plugins install growth
claude plugins install marketing
claude plugins install copywriting
claude plugins install development
claude plugins install review
claude plugins install ux
```

## Usage

Skills are invoked via the Skill tool in Claude Code:

- `seo:seo-optimization`
- `copywriting:copywriting-orchestrator`
- `development:flutter-workflow`
- etc.
