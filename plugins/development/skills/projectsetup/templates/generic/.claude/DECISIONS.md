# Architektonická rozhodnutí

Tento soubor obsahuje důležitá rozhodnutí učiněná během vývoje projektu.

**Kdy logovat:**
- Volba mezi 2+ validními přístupy
- Rozhodnutí ovlivňující více souborů
- Dlouhodobě platná rozhodnutí
- Cokoliv, na co by se budoucí já mohl ptát "proč?"

**Formát:**
```
## [YYYY-MM-DD]: [Název rozhodnutí]
**Kontext:** [Proč bylo potřeba rozhodnout]
**Rozhodnutí:** [Co jsme zvolili]
**Alternativy:** [Co jsme zvažovali]
**Důvod:** [Proč tato volba]
```
