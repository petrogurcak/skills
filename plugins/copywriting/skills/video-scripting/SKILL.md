# Video Scripting

## Overview

Long-form video lives or dies on **retention curve**. Three layered frameworks combined:

1. **Tactical retention data** (Video Creator Playbook + MrBeast/Veritasium/Ali Abdaal benchmarks) — what keeps viewers watching second-by-second.
2. **Snyder 15-Beat Sheet** (Save the Cat) — narrative structure scaled to video length.
3. **Heath SUCCESs** (Made to Stick) — psychology layer for memory + emotion.

**Core principle:** Hook → Substance → Payoff. If the payoff is missing, the algorithm punishes you and trust breaks.

**Announce:** "I'm using video-scripting to engineer retention and structure."

## When to Use

**USE:**

- Long-form YouTube (10-30 min)
- VSL (Video Sales Letter) — short 8-15 min or long 45-90 min
- Course module videos
- Explainer / educational videos
- Sales/pitch videos over 60s

**DON'T USE:**

- Short-form TikTok / Reels / Shorts under 60s → `copywriting:ig-content`
- Written sales copy → `copywriting:web-copy`
- Pure storytelling without video format → `copywriting:storytelling`

## Hook Layer (First 5-7 Seconds)

The first 5-7 seconds decide everything. If retention drops below 70% here, the video is dead.

**Snyder Logline (4 elements compressed into hook):**

1. **Irony** — paradox or contradiction
2. **Mental picture** — concrete visual
3. **Audience + cost** — who it's for and what's at stake
4. **Killer title** — promise in one line

**Heath Curiosity Gap:** State what viewer doesn't know but should. "Most people get X wrong — here's why."

**Auditory + Visual hook combination:** First 2s = strong visual (pattern interrupt, motion, face). First 5s = verbal hook delivering promise.

**Top hook templates (from Video Playbook 50-list):**

1. **Bold claim** — "I made $1M doing X in 30 days."
2. **Contrarian** — "Everything you've been told about X is wrong."
3. **Curiosity gap** — "Nobody talks about this one thing that..."
4. **Question hook** — "Why do 90% of [audience] fail at X?"
5. **Stakes hook** — "If you do X, you'll lose Y."
6. **Result reveal** — Show end state first, then "Here's how I got here."
7. **Story cold open** — Drop into mid-scene, then "Let me back up..."
8. **Pattern interrupt** — Unexpected visual/sound in first 1s
9. **Listicle promise** — "5 things that doubled my X"
10. **Mistake hook** — "I wasted 2 years doing this — don't repeat my mistake."
11. **Authority hook** — "After [credential], here's what I learned."
12. **Visual demonstration** — Show the thing working before explaining
13. **Time-bound** — "In the next 10 minutes you'll learn..."
14. **Negation hook** — "This isn't another [generic topic] video."
15. **Direct callout** — "If you're [specific audience], stop scrolling."

## Substance Layer (75-80% of body)

- **Hook → Substance → Payoff** — every segment honors the loop. Open loop, fill with substance, close.
- **Edutainment (Laugh or Learn)** — every 30-60s the viewer should either learn or laugh. If neither, cut.
- **Heath SUCCESs:**
  - **Concrete > Abstract** — show "$47 grocery bill" not "saved money"
  - **Velcro memory** — hook new info to what viewer already knows
  - **Sinatra Test** — one undeniable proof beats five generic claims
  - **Mother Teresa effect** — one specific person > statistics
- **Pattern interrupts every 3-7s** — cut, zoom, text overlay, sound effect, B-roll. Retention data: +23% with consistent interrupts vs static talking-head.
- **B-roll** for every abstract claim, **text overlays** for retention spikes, **sound effects** for punctuation.

## Payoff + CTA (Last 5-10s)

- **Close every open curiosity loop** opened in the hook
- **CTA fast** — under 10s, one ask only (subscribe / link / next video)
- **End immediately** after CTA — drag = retention drop on next video

**Warning:** Payoff missing → trust break + algorithm penalty (watch-time + return-viewer score both drop).

## VSL Specifics

**Jon Benson 5-step VSL structure:**

1. **Problem** — agitate the pain (maps to Snyder Catalyst + Debate)
2. **Why nothing works** — disqualify alternatives (Bad Guys Close In)
3. **The secret** — reveal mechanism (Midpoint + Break Into Three)
4. **Proof** — testimonials, data, demos (Finale build-up)
5. **Offer** — pricing, bonuses, scarcity, CTA (Finale + Final Image)

**Snyder All Is Lost = "agitate problem" phase** — the moment where viewer realizes status quo is unsustainable.

**Length rule:**

- **Long VSL (45-90 min)** for high-ticket >$1k offers
- **Short VSL (8-15 min)** for <$1k offers and mid-funnel
- Under $200 → don't use VSL, use direct copy

## Common Mistakes

| Mistake | Fix |
| --- | --- |
| **"Pope in the Pool" missed** — boring exposition kills retention | Disguise setup with visual action / B-roll / demonstration |
| **Inactive Hero** — narrator/protagonist passive | Hero must make decisions and act, not just observe |
| **Talking the Plot** — narrator describes instead of showing | Show Don't Tell — demonstrate on screen |
| **Too Much Pipe** — 40-min Minority Report problem, setup eats body | Trim setup to <15% of runtime |
| **Missing Payoff** — open loop never closed | Map every promise in hook to a payoff beat |
| **Wrong genre** — educational dressed as Action | Use Snyder genre map above |
| **Static talking-head** — no pattern interrupts | Cut every 3-7s, B-roll every abstract claim |
