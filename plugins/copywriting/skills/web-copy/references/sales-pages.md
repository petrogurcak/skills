# Sales & Landing Pages

**CRITICAL: Determine audience awareness stage FIRST.**

Sales/landing page structure depends entirely on awareness level:

- **Unaware/Problem Aware:** Need long-form education before pitch
- **Solution/Product Aware:** Can use direct persuasion frameworks
- **Most Aware:** Short-form, remove friction only

**See:** `FRAMEWORKS.md` - Eugene Schwartz 5 Awareness Stages

---

## 5-Step Blueprint

**Source:** Web Copy That Sells (Maria Veloso)
**Best for:** Solution Aware -> Product Aware audiences

### 5 Questions Framework

- [ ] Odpovedet na vsech 5 otazek
- [ ] Rozsirit emocemi a storytellingem
- [ ] Pridat duveryhodnost a zaruky

1. **Co je problem?** - Hledani bolesti, problemu nebo dilematu
2. **Proc problem nebyl vyresen?** - Identifikace neuspesnych pokusu
3. **Co je mozne?** - Popis idealniho stavu po vyreseni
4. **Co je ted jine?** - Jak vas produkt problem nove resi
5. **Co byste meli udelat nyni?** - Jasna vyzva k akci

**Rozpracovani:** Po sestaveni Blueprintu vstrikni emoce, pridej bullet points, bonusy + zaruku, prvky duveryhodnosti, nahrad racionalni slova emocnimi.

## Triada/Trifecta Principle

**Source:** Web Copy That Sells + Piste jako copywriter III. dil

### 3-Phase Persuasion

- [ ] Faze 1: REZONANCE (Identifikace)
- [ ] Faze 2: REDEFINICE (Premena)
- [ ] Faze 3: REZISTENCE (Vzdor)

> "Trojice je nejefektivnejsi zkratkou pro rychle obsahnuti celku."

**REZONANCE** - Vytvoreni emocionalniho souzneni. Ukazte, ze rozumite jejich problemu.

**REDEFINICE** - Poskytnuti nove perspektivy. Argumenty, ktere donuti videt veci v novem svetle.

**REZISTENCE** - Byt rychlejsi nez jejich podvedomi. Predvidejte namitky a zapaste pochyby drive, nez prijdou.

**Typy argumentu:** Kvantitativni ("89 % klientu spolupracovalo opakovane"), Narativni ("Mam rad, kdyz si k zadani vzdy sedneme..."), Logicky ("Pracuji ve velke agenture a znam cely proces...")

## PROTTO Steps 4-6

**Source:** PROTTO Desatero 2. dil

### Step 4: PROBud (Awaken Attention)

- [ ] Vytvorit headline, ktery zaujme za 3 sekundy
- [ ] Pouzit jednu ze 3 strategii
- [ ] Sladit prvni vety s nadpisem

**3 strategie:** Otevrena otazka, Osloveni ctenare, Obrazne psani

> "Sladte prvni vety textu s nadpisem. Obsah musi dodrzet slib, ktery nadpis ucinil."

### Step 5: PRObodni (Emotional Impact)

- [ ] Identifikovat hlavni emocni spinac
- [ ] Pouzit pribehy a realne situace
- [ ] Zapojit vsechny smysly

> "EMOCE jsou kovova spicka sipu, ktery pronika rozumovym brneni ctenare."

Proces: Identifikace emocniho spinace (strach/radost/uleva/hrdost/touha) -> Konkretizace pribehy -> Vizualizace smysly.

### Step 6: PROlom (Break Resistance)

- [ ] Pridat reference a testimonialy s fotkami
- [ ] Uvest certifikaty/oceneni
- [ ] Garance a zaruky
- [ ] Fakta a cisla
- [ ] Priznat omezeni (autenticita!)

> "Zakaznici pochybuji. Vasim ukolem je rozprasit jejich obavy pomoci dukazu duveryhodnosti."

**Mocny trik:** Priznani omezeni produktu posiluje duveru.

## Aggressive Direct Copy (Optional)

**Source:** Piste jako pornomagnat

**Kdy pouzit:** Pri tvorbe ostreho copy, kdy se na webu lame konverzni chleba.

**Techniky:** "Bigger and better", "Meet real people in your area", "Don't wait", "No bullshit"

**Kdy NEPOUZIVAT:** B2B formalni prostredi, konzervativni odvetvi, kdyz to znacka neunese.

## Psychological Triggers

| Trigger | Princip | Aplikace na web |
|---------|---------|-----------------|
| **Dar** | Reciprocita | E-book zdarma na homepage |
| **Dukaz** | Socialni tlak | "347 spokojenych klientu" |
| **Triada** | Magicka trojice | Rezonance -> Redefinice -> Rezistence |
| **Zeigarnik** | Nedokoncene ukoly | Cliffhanger na konci stranky |
| **Emoce** | Prava hemisfera | Emocni slova misto racionalnich |
| **Kotva** | Relativni ceny | Cenove srovnani na landing page |
| **Zapojeni** | Psani = zavazek | Kvizy, kalkulacky, formulare |
| **Cyrano** | Emocni obal | Pribeh znacky na About Us |
| **Hypnoza** | NLP techniky | Vlozene prikazy v textu |
| **Limit** | Nedostatek | "Posledni mista!", "Do konce tydne" |
