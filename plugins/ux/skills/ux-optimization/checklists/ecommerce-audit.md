# E-commerce Audit Checklist

Use this for product pages, cart, and checkout.

## Product Photos
- [ ] Min 3 photos (feature, angle, scale)
- [ ] Ideal 5-8 photos with zoom
- [ ] Scale reference included
- [ ] Zoom available (2Ã— minimum)
- [ ] Mobile-optimized gallery
- [ ] Fast loading (WebP, lazy load)

## Cart Confirmation
- [ ] Immediate feedback (< 100ms)
- [ ] Visual confirmation (toast or slide-in)
- [ ] Cart count updated
- [ ] Option to view cart or continue shopping
- [ ] Works on mobile

## AOV Optimization
- [ ] Free shipping threshold visible
- [ ] Progress bar to free shipping
- [ ] Bundle offers (min 10% savings)
- [ ] Cross-sell in cart (relevant products)
- [ ] Quantity discounts displayed

## Product Detail
- [ ] Clear pricing (no hidden costs)
- [ ] Availability shown
- [ ] Shipping info visible
- [ ] Trust badges (returns, security)
- [ ] Reviews visible
- [ ] Clear CTA ("Add to cart", not "Buy")

## Expected Impact
- Product photos: +20-40%
- Cart confirmation: +10-20%
- AOV strategies: +15-35%

Total potential: +45-95% improvement
