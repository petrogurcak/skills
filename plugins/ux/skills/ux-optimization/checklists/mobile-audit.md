# Mobile UX Audit Checklist

60%+ traffic is mobile. Audit with this checklist.

## Touch Targets
- [ ] All buttons ≥ 48×48px
- [ ] 8px spacing between targets
- [ ] Important actions in thumb zone (bottom)
- [ ] Less important in hard-to-reach (top corners)

## Forms
- [ ] Font size ≥ 16px (prevents zoom)
- [ ] inputmode attributes set correctly
- [ ] Touch-friendly height (≥ 48px)
- [ ] Labels above fields (always visible)
- [ ] Sticky submit button
- [ ] Progress indicator (multi-step)
- [ ] Autosave (localStorage)

## Performance
- [ ] Lazy loading for images
- [ ] WebP format with fallback
- [ ] Fast load (< 3 seconds)
- [ ] No layout shifts (CLS < 0.1)

## Navigation
- [ ] Easy to use with one hand
- [ ] Back button works
- [ ] Search easily accessible
- [ ] Cart icon visible

## Testing
- [ ] Tested on real iOS device
- [ ] Tested on real Android device
- [ ] Tested in both portrait and landscape
- [ ] Tested with slow 3G connection

## Expected Impact
- Touch targets: +20-35%
- Form optimization: +25-40%
- Performance: +15-30%

Total potential: +60-105% mobile improvement
